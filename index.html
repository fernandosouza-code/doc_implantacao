Matriz de comunicação
▾
?
Como preencher esta seção?
Matriz de comunicaçãoDefine como as informações do projeto serão comunicadas — quem precisa saber o quê, quando e por qual canal.
  • Informação/assunto: o que será comunicado (ex: status diário, alerta de risco)
  • Para quem: o destinatário da comunicação do lado do cliente
  • Quando: frequência ou gatilho (ex: diário, imediato no desvio)
  • Como: canal utilizado (ex: Grupo WhatsApp, e-mail, reunião)
  • Responsável: quem da Cobli faz a comunicação

Como preencher esta seção?
Levantamento da frotaDefine o escopo técnico — base para custo, prazo e alocação de técnicos.
  • Total e cancelados: controla a quantidade real de instalações
  • Tipo/modelo: cada modelo pode exigir instalação diferente
  • Customizações: tudo que foge do padrão — alinhe antes com os técnicos

Como preencher esta seção?
Bases de operaçãoRegistra onde as instalações acontecerão — essencial para logística de envio.
  • ID da base: nome de referência (ex: Matriz, Base SP)
  • Endereço: necessário para envio de equipamentos e deslocamento
  • Responsável local: pessoa do cliente que acompanha o checklist — obrigatório
  • Qtd. de veículos: define alocação de técnicos e tempo em cada base

Como preencher esta seção?
Riscos e premissasAntecipar problemas antes que aconteçam.
  • Risco: algo que pode atrapalhar (ex: chuva, no-show, veículos não liberados)
  • Premissa: condição necessária para o projeto funcionar
  • Impacto: o que acontece se o risco se concretizar
  • Mitigação: o plano B
